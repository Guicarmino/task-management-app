# Cloud-Based Task Management Application

## Overview
This is a basic prototype of a cloud-based task management application built with React and Firebase.

## Implemented Features
- User registration
- User login
- Task creation
- View tasks by authenticated user
- Logout

## Technologies Used
- React
- Vite
- Firebase Authentication
- Cloud Firestore

## How to Run
1. Open terminal in the project folder
2. Run `npm install`
3. Run `npm run dev`
